● Code & Artifact 📄 Traceability

Temporal KG question generation is a constraint-preservation problem. A fluent question can still be wrong when it reverses before/after, drops first/last, changes the reference event, or asks for the wrong answer field.

Explicit intent Temporal rule, answer field, reference point, graph shape, and time granularity are represented before generation.
Indexed pattern memory Typed question patterns are retrieved by temporal compatibility before relation and topology similarity.
Constraint validation Drafts are checked for leakage, unresolved placeholders, temporal mismatch, and answer-field drift.

Method

From graph evidence to a controlled question

  1. 01Construct the Temporal Intent State.
  2. 02Retrieve compatible typed patterns.
  3. 03Adapt patterns to the input graph.
  4. 04Validate constraints and realize the question.
