Motivation

Fluent questions still fail when they flip before/after, drop first/last cues, shift the reference event, or ask for the wrong answer field.

Temporal Intent State Answer field, temporal rule, reference point, graph context.
Indexed Pattern Memory Typed question patterns retrieved by temporal compatibility.
Constraint Validation Checks answer leakage, placeholder resolution, rule drift, answer-field drift.
